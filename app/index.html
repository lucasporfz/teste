Validador — comparar calculadora com hunt real

Cole o conteúdo do server log de uma hunt real do Tibia. A ferramenta extrai automaticamente os parâmetros (XP/h, dano, taxa de charm, tamanho da box) e compara com o que a calculadora prevê com os mesmos parâmetros. Se os números baterem dentro de uma margem razoável, o modelo está validado empiricamente.
